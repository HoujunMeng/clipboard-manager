## 技术决策记录

1. **WPF + WinForms 混合**：`UseWindowsForms=true` 以使用 `NotifyIcon`（最成熟的托盘方案）
2. **Win32 P/Invoke**：`GetClipboardSequenceNumber()` 代替事件（与 Mac 版轮询模式一致）
3. **窗口隐藏而非关闭**：`OnClosing` 中 `e.Cancel=true` + `Hide()` 保留面板状态
4. **图标备选方案**：.ico 不存在时动态绘制剪贴板图标（`CreateFallbackIcon()`）

## 遇到的问题

1. **brew 安装 .NET SDK 需 sudo** → 用 dotnet-install.sh 安装到 `~/.dotnet/`
2. **macOS 无 WPF 工作负荷** → 手写 .csproj + 所有文件，Windows 上编译
3. **macOS 无 Windows 编译器验证** → 代码逻辑完整，需在 Windows 上实际编译运行
