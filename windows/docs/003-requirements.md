# 003 — 需求文档

**项目**：历史粘贴板 Windows 版
**版本**：1.0.0
**创建日期**：2026-07-22

---

## 概述

「历史粘贴板」是一款 Windows 系统托盘剪贴板管理工具。自动记录用户复制过的文字和图片，支持搜索、置顶和再次粘贴。所有数据存储在本地。

## 用户故事

### US-01：自动记录剪贴板
> 作为用户，我希望复制任何文字或图片后，应用自动保存这条记录，无需我手动操作。

- 文字支持 UTF-8/Unicode
- 图片支持：剪贴板内 BMP/DIB/PNG + Finder 复制的图片文件（PNG/JPG/GIF/BMP/WEBP）
- 重复内容不重复保存（SHA256 去重）
- 应用本身写回剪贴板不触发保存

### US-02：浏览历史记录
> 作为用户，我希望点击任务栏图标后弹出历史面板，查看我复制过的所有内容。

- 面板宽 380px × 高 520px
- 卡片列表显示预览，图片显示缩略图
- 置顶内容排在最前
- 每条记录显示相对时间（刚刚 / N分钟前 / N小时前 / N天前 / 日期）

### US-03：搜索历史
> 作为用户，我希望在面板顶部的搜索框输入关键词，实时过滤历史记录。

- 支持中文/英文搜索
- 输入即搜，无需按回车
- 清空搜索框恢复完整列表

### US-04：重新复制
> 作为用户，我希望点击某条历史记录后，内容自动复制到剪贴板，面板关闭。

- 单击文字卡片 → 文字写入剪贴板
- 单击图片卡片 → 图片写入剪贴板
- 复制成功后面板自动关闭

### US-05：置顶内容
> 作为用户，我希望把常用的内容固定在顶部，不受过期清理影响。

- 点击图钉图标切换置顶/取消置顶
- 置顶卡片有蓝色左侧标识和淡蓝背景
- 置顶内容不会被自动清理

### US-06：删除记录
> 作为用户，我希望删除不需要的历史记录。

- 点击卡片右上角 × 删除记录
- 图片记录删除时同时删除磁盘文件
- 删除不可撤销（与 Mac 版一致）

### US-07：设置保留天数
> 作为用户，我希望设置历史记录的保留时间（1/3/5 天），过期自动清理。

- 默认 3 天
- 分段选择器：1 天 / 3 天 / 5 天
- 仅清理非置顶记录
- 更改保留天数时立即触发清理

### US-08：开机自启
> 作为用户，我希望电脑开机后应用自动启动，无需手动打开。

- 开关位于设置窗口内
- 默认开启
- 通过注册表 Run 键实现

### US-09：系统托盘操作
> 作为用户，我希望通过任务栏图标进行操作。

- 左键单击托盘图标 → 切换面板显示/隐藏
- 右键单击托盘图标 → 弹出菜单（设置 / 关于 / 退出）
- 图标始终可见（如果用户允许）

## 非功能需求

| 类别 | 需求 |
|------|------|
| 性能 | 面板打开 <200ms，搜索响应 <100ms |
| 内存 | 空闲时 <50MB RAM |
| 存储 | 数据库大小 <10MB（200 条记录以内） |
| 隐私 | 数据 100% 存储在本地，无网络请求 |
| 兼容性 | Windows 10 1809+ / Windows 11 |
| 语言 | 简体中文 UI |
| 数据库兼容 | 与 Mac 版相同 schema，可互相拷贝使用 |

## 与 Mac 版差异

| 差异点 | Mac 版 | Windows 版 | 原因 |
|--------|--------|------------|------|
| 托盘位置 | 右上角菜单栏 | 右下角任务栏 | 系统规范不同 |
| 面板弹出方式 | NSPopover 原生浮动层 | 无边框 Window 模拟 | WPF 无 Popover 控件 |
| 面板关闭方式 | 点击外部自动关闭 + transient 行为 | 监听 Deactivated 事件 | 实现方式不同，行为相同 |
| 开机自启 | SMAppService / LaunchAgent | 注册表 Run 键 | 系统 API 不同 |
| 图标 | SF Symbol `doc.on.clipboard` | 自定义 .ico 文件 | 无 SF Symbol |
| 文字选取 | macOS 13+ textSelection 支持 | WPF TextBox 默认支持 | 框架原生支持 |
| 安装方式 | DMG 磁盘镜像 | Inno Setup .exe 安装器 | 平台分发方式不同 |
| 发行渠道 | GitHub Release DMG | GitHub Release .exe 安装器 | 一致 |
